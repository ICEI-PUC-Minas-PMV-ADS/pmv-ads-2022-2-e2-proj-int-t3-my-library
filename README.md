# PROJETO: DESENVOLVIMENTO DE UMA APLICAÇÃO INTERATIVA

`Tecnologia em Análise e Desenvolvimento de Sistemas`

`Projeto Eixo 2`

`Segundo semestre de 2022`

Está sendo desenvolvido um sistema de biblioteca pessoal.

O usuário conseguirá organizar seu acervo pessoal, bem como solicitar empréstimos a outros usuários e também consultar suas respectivas bibliotecas.
